# 專案名稱：羽球流光 (Badminton Flow) 報名系統

> **The Vibe:** 前端介面要簡潔流暢，使用淺色背景。操作要快、狠、準，像扣殺一樣流暢。管理後台要像「儀表板」一樣精準且高效。欄位調整應具備彈性，讓非技術人員也能透過 Google Sheet 輕易改動。

### 1. 核心願景
* **管理自動化**: 透過「設定頁」直接控制前端顯示的場次、時段與欄位。
* **動態表單**: 前端表單應根據後台定義的「額外欄位」自動渲染，無需重複改寫代碼。
* **極速報名**: 使用者只需輸入姓名與聯絡方式（line或電話）以及下拉式選單的輸入方式，3秒內完成報名。
* **透明化**: 即時顯示剩餘名額，充滿「即將額滿」的動態感。

### 2. 核心功能
- [ ] **多場次管理**: 支援選擇不同日期與時段，並各自計算名額。
- [ ] **動態欄位系統**: 後台可自定義「聯絡窗口」、「繳費資訊」等欄位，前端自動同步。
- [ ] **下拉式導向**: 程度、場次、時段均使用流暢的下拉選單。
- [ ] **報名人數加總**: 報名時可選擇人數（1人、2人等，上限5人），系統需自動累加至該場次總額。
- [ ] **動態報名表單**: 包含姓名、電話、程度（初級/中級/高級）、報名場次、報名時段、報名人數、備註。
- [ ] **名額即時計算**: 從 Google Sheet 讀取已報名人數，顯示「已報名 8/12 人」。
- [ ] **自動防呆**: 當名額滿了，自動將報名按鈕變為灰色「候補中」。
- [ ] **後端紀錄**: 所有資料透過 GAS 寫入 Google Sheet。

### 3. 欄位定義
* **固定欄位**: 姓名、電話、場次、時段、程度（下拉）、報名人數、備註。
* **動態擴充**: 繳費帳號、窗口備註、裝備租借需求等（由後台 Sheet 決定）。

### 4. 視覺與體驗
* **配色**: 背景 `#0f172a` (深藍黑)，主色 `#d9f99d` (螢光綠)。
* **字體**: 粗體標題，數字使用 Monospace 感，強調競技感。
* **動畫**: 提交成功後，畫面上要出現一個「羽毛球飛過」的轉場效果。
